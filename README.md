## 🧩 프로젝트 소개

| 항목          | 내용                                                         |
| ------------- | ------------------------------------------------------------ |
| 📌 프로젝트명 | EmotionLog                                                   |
| 🗓️ 개발 기간  | 2025.02 ~ 진행 중 (기능 추가 및 개선 계속 진행)              |
| 🛠️ 주요 기술  | React, TypeScript, Supabase, Redux Toolkit, Recharts         |
| ✨ 주요 기능  | 감정 일기, 통계 대시보드, 감정별 테마, 필터, 페이지네이션 등 |

- ✍️ 감정 일기 작성, 파일 첨부 기능
- 📊 감정 통계 대시보드 (도넛 차트, 막대 그래프, 워드 클라우드, 감정 분석 등)
- 📅 캘린더 필터, 통계 기간 필터, 감정 태그 필터, 페이지네이션
- 📌 로그인 기반 개인 감정 히스토리 저장
- 💬 감성적인 UI/UX (디자인에 따라 변화하는 테마, 컬러 시스템 적용)

**EmotionLog**는 하루에 하나의 감정을 기록하고, 해당 데이터를 기반으로 감정 흐름과 통계를 제공하는 감정일기 플랫폼입니다.

이 프로젝트는 단순한 기능 나열이 아닌, '감정'이라는 추상적인 데이터를 시각화하고 사용자 경험을 감성적으로 구성해보자는 목적에서 출발했습니다.

'기획 → 구조 설계 → 상태 관리 → UI 흐름 → 통계 분석 → 배포'까지 전 과정을 실제 서비스처럼 구축하는 데 중점을 두었습니다.

**사용자 흐름의 예시**

1. 사용자가 로그인하면, 이전 감정 데이터를 기반으로 오늘의 감정을 입력하도록 유도합니다.
2. 작성된 감정은 자동으로 태그 분류 + 날짜 연동되어 캘린더와 통계에 반영됩니다.
3. 사용자는 감정 흐름을 전체적인 컬러 테마에서 볼 수 있고, 대시보드에서 시각화해보고, 원하는 감정을 필터링하거나 분석할 수 있습니다.
